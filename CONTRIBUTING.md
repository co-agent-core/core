# 贡献指南

感谢你愿意为 Co-Agent Core 贡献代码与思想！

## 行为准则

本项目采用 [贡献者公约](https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/)。
我们期待所有参与者相互尊重，建设性地讨论，共同维护一个开放、友好的社区。

## 贡献者许可协议 (CLA)

本项目由 **江门市戴梦达管理咨询有限公司** 主导并维护。为保证项目版权归属清晰，以及未来商业化的可行性，所有贡献者必须同意以下条款：

1. 你授予 **江门市戴梦达管理咨询有限公司** 一份永久的、免费的、全球范围内的、不可撤销的版权许可，允许其在 Co-Agent Core 项目（包括其商业版本）中使用、修改、分发你的贡献。

2. 你保留对你所提交内容的原始所有权和署名权。

3. 你保证你拥有所提交内容的合法权利，并且你的贡献不侵犯任何第三方的知识产权或其他合法权益。

4. 如果你的贡献是基于你受雇期间的工作成果，你保证你的雇主已知悉并同意此许可协议。

### 如何签署 CLA

在提交第一个 Pull Request 时，请在 PR 描述中明确写下以下声明：

> 我已阅读并同意 Co-Agent Core 项目的贡献者许可协议 (CLA)。我保证我拥有所提交内容的合法权利，并授权江门市戴梦达管理咨询有限公司在上述协议条款下使用我的贡献。

## 我们寻找的贡献者

Co-Agent Core 不只是一个技术项目。我们寻找的贡献者包括但不限于：

| 角色 | 贡献方式 |
|------|----------|
| **思想贡献者** | 提供管理原则、人机协同模式、真实案例 |
| **领域贡献者** | 定义行业 DSL 模板、最佳实践、业务规则 |
| **模型贡献者** | 适配国产大模型、优化模型连接器 |
| **代码贡献者** | 改进引擎核心、修复 Bug、编写测试 |
| **文档贡献者** | 完善文档、翻译、撰写教程 |
